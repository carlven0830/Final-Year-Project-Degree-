1.Please place the code file in the python path, or else you cannot run it.
2.If you did not download python in your window platform, please download it. This is the download link for the window platform. 
https://www.microsoft.com/store/productid/9PJPW5LDXLZ5?ocid=pdpshare
3.After installing python in your window, you can find your python path like "C:\Users\user\AppData\Local\Packages\PythonSoftwareFoundation.Python.3.10_qbz5n2kfra8p0\LocalCache\local-packages\Python310\Scripts"
4.Then place the python code here and you can run it.
